# React + Vite Using Bun

```sh
bun create vite .
bun install
bun run dev
```

## Install dependencies

```sh
bun add @solana/wallet-adapter-base @solana/wallet-adapter-react @solana/wallet-adapter-react-ui @solana/wallet-adapter-wallets @solana/web3.js
```
